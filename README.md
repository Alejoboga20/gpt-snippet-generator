# GPT Code Snippets Generator

This is a simple CLI tool to generate code snippets from a given text using GPT.

## Installation

```bash
poetry install
```

Fill the environment variables in `.env` file.

```
OPENAI_API_KEY=
```

## Usage

1. Initiate the virtual environment

```bash
poetry shell
```

2. Run the CLI

```bash
python main.py
```
